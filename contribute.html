Jazz Vinyl Guide is a collector reference for people who care about listening to the best versions of the greatest jazz albums. Each guide covers the recording history, original pressing identification, the reissue hierarchy with verified catalogue numbers, and an honest buyer's guide by budget.

Contribute

Help improve these guides

Every guide on this site is a living document. If you own a pressing we haven't covered, have a direct A/B comparison to share, or spot an error in a catalogue number — we want to hear from you.

How to edit a page

Each album page has an Edit this page on GitHub button at the bottom. Clicking it takes you to the source file on GitHub where you can propose a change directly in the browser — no technical knowledge required. GitHub will ask you to create a free account if you don't have one.
