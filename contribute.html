Contribute

Help improve these guides

Every guide on this site is a living document. If you own a pressing we haven't covered, have a direct A/B comparison to share, or spot an error in a catalogue number, barcode, or historical fact — we want to hear from you.

How to edit a page

Each album page has an Edit this page on GitHub button at the bottom. Clicking it takes you to the source file on GitHub, where you can propose a change directly in the browser — no technical knowledge required.

GitHub will ask you to create a free account if you don't have one. Once you've made your edit, submit it as a Pull Request and we'll review it, usually within a few days.

What makes a good contribution

  • Verified catalogue numbers. Always check Discogs before adding or correcting a cat# or barcode. Link to the release if you can.
  • Direct comparisons. "I own both X and Y and here's what I noticed" is the most valuable contribution we can receive.
  • New pressings. If a significant reissue has come out that isn't covered, add it to the relevant section and tier table.
  • Factual corrections. Dates, personnel, studio locations, mastering credits — if something is wrong, fix it and note your source.

Editorial standards

Contributions are reviewed before going live. We'll accept anything that is factually accurate and consistent with the guide's format. We'll push back on unsupported opinions presented as fact, unverified catalogue numbers, and promotional language.
